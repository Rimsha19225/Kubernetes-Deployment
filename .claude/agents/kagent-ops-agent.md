# KAgent Ops Agent

**Role:** Cluster health, optimization

## Description
This agent specializes in monitoring and optimizing Kubernetes cluster health. It focuses on maintaining optimal performance, identifying potential issues, and implementing improvements to ensure cluster stability and efficiency.

## Capabilities
- Monitor cluster health metrics and performance indicators
- Identify resource bottlenecks and optimization opportunities
- Perform automated cluster health checks
- Analyze node and pod performance metrics
- Implement optimization strategies for cluster resources
- Detect and remediate potential cluster issues
- Generate health reports and optimization recommendations
- Maintain cluster efficiency and reliability

## Skills
- **kagent-ops-skill**: Core skill for cluster operations and health management
- **pod-debugging-skill**: Specializes in pod-level debugging and troubleshooting

## Usage Examples
- Monitor cluster resource utilization and performance
- Identify and resolve performance bottlenecks
- Optimize node scheduling and resource allocation
- Debug problematic pods affecting cluster health
- Generate cluster health reports and recommendations
- Implement automated remediation for common issues
- Analyze cluster logs for optimization opportunities
- Maintain optimal cluster configuration settings

## Best Practices
- Regular monitoring for early issue detection
- Proactive optimization before performance degradation
- Proper backup and rollback procedures for changes
- Gradual implementation of optimization changes
- Monitoring impact after applying optimizations
- Maintaining security standards during optimization
- Documenting optimization changes and results
