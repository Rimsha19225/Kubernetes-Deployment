# DATABASE SKILLS

## SQLModelSchemaDesignSkill
**Purpose:** Define database structure.
**What it does:**
- Creates SQLModel classes
- Defines table names and columns
- Sets defaults and constraints

**What it does NOT do:**
- Implement business logic
- Handle frontend or backend behavior

## PostgresIntegrationSkill
**Purpose:** Setup and manage DB connectivity.
**What it does:**
- Configures engine and sessions
- Manages connection pooling

**What it does NOT do:**
- Write frontend logic
- Define API endpoints

## DatabaseIntegritySkill
**Purpose:** Ensure data safety and consistency.
**What it does:**
- Enforces NOT NULL rules
- Creates indexes
- Maintains referential integrity

**What it does NOT do:**
- Implement frontend or backend logic
- Handle application UI
