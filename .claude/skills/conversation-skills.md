# CONVERSATION SKILLS

## Multi-Step Conversation Skill

**Skill Name:** multi-step-conversation
**Description:** Handle commands like "first do this, then do that"
**Function:** Process and execute sequential commands within a single user request
**Input:** Multi-step command sequence from user
**Output:** Execution results for each step in the sequence

## Clarification Handling Skill

**Skill Name:** clarification-handling
**Description:** Ask questions for ambiguous commands
**Function:** Identify unclear or ambiguous user requests and seek clarification to ensure accurate execution
**Input:** Ambiguous user command
**Output:** Clarifying questions presented to user for resolution
