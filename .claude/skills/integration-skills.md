# INTEGRATION SKILLS

## APISchemaContractSkill
**Purpose:** Ensure frontend and backend are in sync.
**What it does:**
- Aligns request and response structures
- Detects breaking changes in API contracts

**What it does NOT do:**
- Implement APIs
- Write frontend code

## AuthFlowValidationSkill
**Purpose:** Validate end-to-end authentication flows.
**What it does:**
- Tests login → token → API workflows
- Checks expired and invalid token cases

**What it does NOT do:**
- Implement authentication logic
- Handle UI
