# RESPONSE SKILLS

## Response Formatting Skill

**Skill Name:** response-formatting
**Description:** Format lists, confirmations, and summaries
**Function:** Transform raw data into well-formatted, readable responses for users
**Input:** Raw data or operation results
**Output:** Formatted response suitable for user consumption (lists, confirmations, summaries)

## Error Explanation Skill

**Skill Name:** error-explanation
**Description:** Explain errors in simple, user-friendly language
**Function:** Translate technical errors into clear, understandable messages for users
**Input:** Technical error message or exception
**Output:** User-friendly explanation of the error and potential solutions
