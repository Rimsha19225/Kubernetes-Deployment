# FRONTEND SKILLS

## NextJSAppRouterSkill
**Purpose:** Define and structure the frontend application.
**What it does:**
- Sets up layouts, pages, and routing
- Decides which components run on server vs client

**What it does NOT do:**
- Write backend code
- Handle business logic

## TailwindUISkill
**Purpose:** Ensure consistent and responsive UI styling.
**What it does:**
- Creates responsive layouts
- Maintains design consistency
- Uses utility-first styling with Tailwind

**What it does NOT do:**
- Implement application logic
- Handle authentication or backend

## JWTClientHandlingSkill
**Purpose:** Manage frontend authentication.
**What it does:**
- Defines token storage strategy
- Attaches JWT tokens to requests
- Handles logout and token expiry

**What it does NOT do:**
- Verify tokens on backend
- Implement server-side auth logic
