# INTEGRATION & SAFETY SKILLS

## AI Backend Bridge Skill

**Skill Name:** ai-backend-bridge
**Description:** Secure communication between AI and backend
**Function:** Establish and maintain secure communication channels between AI components and backend services
**Input:** AI request requiring backend interaction
**Output:** Securely transmitted request with proper validation and error handling

## Auth Token Propagation Skill

**Skill Name:** auth-token-propagation
**Description:** Forward JWT with AI requests
**Function:** Safely propagate authentication tokens with AI requests to maintain user context
**Input:** User authentication token and AI request
**Output:** AI request with securely attached authentication credentials

## Hallucination Prevention Skill

**Skill Name:** hallucination-prevention
**Description:** Prevent AI from using imaginary data
**Function:** Validate AI responses against real data sources to prevent fabrication of information
**Input:** AI-generated content
**Output:** Verified content with flagged or corrected hallucinated information
